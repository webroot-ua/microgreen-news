# microgreen-news
```
Mobile app for making photo notes.
```
## Google Play
```
https://play.google.com/store/apps/details?id=news.microgreen.photo_notes
```
## Install
```
Run command in console:
git clone https://github.com/webroot-ua/microgreen-news.git
```
## Run
```
cd microgreen-news
npm start
```
